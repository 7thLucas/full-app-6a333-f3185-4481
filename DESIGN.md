# Design Guidelines

## Colors
- Primary: Deep indigo (#3730a3)
- Accent: Gold/teal (#f59e0b or #0d9488)
- Background: Warm white (#fafafa)
- Surface: White (#ffffff)
- Text: Near-black (#111827)

## Typography
- Font: Inter or system sans-serif
- Headings: Bold, clean
- Body: Regular weight, high readability

## Layout
- Left sidebar: toolbar with furniture items (drag source)
- Center: room canvas (main workspace, grid-based)
- Right panel: AI interior designer chat bot
- Top bar: room size controls + app title

## Components
- Grid canvas with snap-to-grid furniture placement
- Resizable, draggable furniture items with labels/icons
- Door/window items that snap to walls
- Chat interface for AI bot with message bubbles
- Room dimension inputs (width × height)

## Elevation & Style
- Flat design with subtle shadows on panels
- Rounded corners (rounded-lg)
- Shadcn/ui components throughout
- Tailwind CSS utility classes
