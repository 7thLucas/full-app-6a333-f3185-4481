# uDesign — Room Designer App

## What it does
Interactive room layout designer with an AI interior designer bot.

## Core Features (MVP)
1. **Room Canvas** — User sets room dimensions (width × length in meters/feet). Canvas renders as a 2D top-down grid.
2. **Furniture Placement** — Drag-and-drop inventory items onto the canvas: chair, table, vase, bed, lamp, door (wall-mounted), window (wall-mounted). Each item has resizable dimensions.
3. **AI Interior Designer Bot** — Chat panel where the bot (powered by an LLM prompt) suggests optimal furniture placement, style tips, and layout improvements based on the current room state.

## Users
Homeowners, renters, and interior design enthusiasts who want to visualize room layouts before buying or rearranging furniture.
